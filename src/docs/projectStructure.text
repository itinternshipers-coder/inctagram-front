# 📁 Структура проекта

Этот документ описывает архитектуру нашего проекта на **Next.js (App Router) + FSD**.

## Основные принципы
- `src/app` — маршрутизация (только layout и pages)
- `entities` — бизнес-сущности (User, Post)
- `features` — действия пользователя (like, auth, edit)
- `widgets` — готовые блоки интерфейса
- `pages` — страницы (собирают фичи и виджеты)
- `shared` — переиспользуемые UI-компоненты и утилиты
